# 🌱 Rediska Day

Аналог сайта Saby «Лучший день — сегодня», весь контент меняется через переменные окружения.

## Деплой на Railway

1. Залей репозиторий на GitHub
2. В Railway: **New Project → Deploy from GitHub repo**
3. В разделе **Variables** добавь нужные переменные (см. ниже)
4. Railway автоматически поднимет сервер

## Переменные окружения

| Переменная | По умолчанию | Описание |
|---|---|---|
| `SITE_TITLE` | `Лучший день — сегодня` | Заголовок на главной |
| `SITE_BRAND` | `🌱 rediska` | Название бренда вверху |
| `THOUGHT_TITLE` | `Мысль дня` | Кнопка и заголовок модалки |
| `THOUGHT_CONTENT` | _(про редиску)_ | Текст мысли дня |
| `PARABLE_TITLE` | `Притча для раздумий` | Кнопка и заголовок |
| `PARABLE_CONTENT` | _(про редиску)_ | Текст притчи (поддерживает HTML) |
| `PRACTICE_TITLE` | `Практика дня` | Кнопка и заголовок |
| `PRACTICE_CONTENT` | _(про редиску)_ | Текст практики |
| `PAST_WEEK_TITLE` | `Прошлая неделя` | Кнопка и заголовок |
| `PAST_WEEK_CONTENT` | _(про редиску)_ | Текст прошлой недели |

## Пример переменных в Railway

\`\`\`
SITE_TITLE=Лучший день — сегодня
THOUGHT_CONTENT=Каждое утро начинается с выбора. Ты выбрал редиску.
PARABLE_CONTENT=Жила-была редиска. Она не знала зачем. Но это её не останавливало. <strong>Мораль: действуй.</strong>
PRACTICE_CONTENT=Съешь сегодня что-нибудь красное и хрустящее.
\`\`\`

## Локальный запуск

\`\`\`bash
npm install
THOUGHT_CONTENT="Моя кастомная мысль" node server.js
\`\`\`
